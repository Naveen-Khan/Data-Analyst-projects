## Project Overview
This project focuses on analyzing sales data from a fictitious coffee shop located in New York City.  
An interactive Excel dashboard was developed to track key business metrics, identify sales trends, and support data-driven decision-making.

## Objectives
- Analyze overall sales performance
- Identify top-selling products
- Understand customer purchasing behavior
- Compare store-wise sales and footfall
- Analyze hourly and weekday sales trends
- Visualize category and size-based sales distribution



## Key Performance Indicators (KPIs)
- Total Sales  
- Total Transactions  
- Average Order per Person  
- Average Bill per Person

## Dashboard Analysis
The dashboard includes the following insights:

### Time-Based Analysis
- Hourly transaction trends
- Orders by weekdays
- Month and day-level filtering using slicers

### Product Analysis
- Top 5 products based on sales
- Category-wise sales distribution
- Size-wise order distribution
